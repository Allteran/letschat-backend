## letschat-backend
Backend side for Letschat application for teamchallenge.io project.  
Tech stack:
- Spring Boot
- MongoDB for saving data
- WebSocket with SimpleBroker + STOMP to transfer messages between users
- Swagger 3.0 for documentation  
- User authentication with JWT token
- User registration with email and confirmation, using Java Mail Sender

## Documentation with OpenAPI
To get access to UI generated by Swagger you should use http://localhost:8200/swagger-doc/swagger-ui/index.html (by default application is running on port 8200)
